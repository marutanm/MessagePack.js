## Run on

### Browser and node-webkit

```js
<script src="lib/MessagePack.js"></script>
<script>
console.log( MessagePack() );
</script>
```

### WebWorkers

```js
importScripts("lib/MessagePack.js");

console.log( MessagePack() );
```

### Node.js

```js
require("lib/MessagePack.js");

console.log( MessagePack() );
```
